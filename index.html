Crypto Tax Buddy

A Flask web app that provides historical 1-minute OHLC data for a cryptocurrency trading pair at a specified date and time by calling Coinbase Pro's API

This project originally started as a command line program in Java that used the Spring REST client to retrieve account balances and 1-minute OHLC data. I then made the same CLI application using Python, after which I created a simple web app with front-end using Flask. It proved to be extremely useful to me as I relied on it heavily while I was completing my taxes.

NBA Shot Chart Builder

A command line program written in Python that uses swar's API client for nba.com to build shot charts for individual NBA players for a specified regular season

All of my sports knowledge went into basketball, and I was curious as to how shot charts are created for NBA players. This project uses a well-documented API client to access stats from nba.com. With the data, I used a plotting library matplotlib to, first, draw a model of an NBA court, and finally plot the x and y coordinates of every shot attempted by a player for a given season.
